# AI Job Market Analyzer

ML project for salary prediction, job market analysis, and skills demand forecasting.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

1. Train the model:
```bash
python train.py
```

2. Run the web app:
```bash
streamlit run app.py
```

## Features

- **Salary Predictor**: Predict salaries based on job parameters
- **Job Market Analysis**: Analyze trends, top skills, and locations
- **Skills Demand Forecast**: See opportunities based on your skills
